# AGENTS.md

## 1. プロジェクトの目的
### 決定事項
- 個人で活動している人たちから、ホームページ制作の問い合わせを獲得する。
- ホームページ制作事業の営業用ホームページとして機能させる。

### 今後調整すること
- 問い合わせ獲得のKPI（数値目標）。

### 未確定事項
- 具体的なCV地点（フォーム送信のみか、SNS/LINE誘導も含むか）。

## 2. このサイトの役割
### 決定事項
- 「営業用サイト」である。
- 同時にポートフォリオの役割も持つ。
- LP/HPの両方に対応できるが、「最適な形を提案するホームページ制作」として見せる。

### 今後調整すること
- ポートフォリオの見せ方の比重。

### 未確定事項
- LP/HPの表現比率（コピーやビジュアルでの比率）。

## 3. 想定ターゲット
### 決定事項
- 確定ターゲット: 料理教室、料理家、トレーナー、トレーニー、格闘家、個人事業主。
- 拡張ターゲット: ヨガインストラクター、ピラティスインストラクター、ダンサー、フィットネス系インフルエンサー、エステティシャン、セラピスト、整体師、フリーランス美容師、英会話講師、コーチング、カウンセラー、オンライン講師、インフルエンサー、モデル、カメラマン、フリーランス（デザイナーエンジニアなど）、起業家（1人〜少人数）。

### 今後調整すること
- どの層を最優先ターゲットとしてメインコピーに反映するか。

### 未確定事項
- 地域や年齢層などの追加セグメント。

## 4. 提供価値コンセプト
### 決定事項
- コンセプト: 最適な形を提案する設計屋。
- 目的に応じて最適なホームページの形を提案する。
- ただ作るだけでなく、設計・導線・見せ方まで考える。

### 今後調整すること
- コンセプトのサブコピーや言い回し。

### 未確定事項
- コンセプトを裏付ける具体的な事例や数字。

## 5. 価格表示方針
### 決定事項
- 価格は細かく分けすぎない。
- LP/HPの細かい価格表をそのまま並べない。
- 表向きの見せ方はシンプルにする。
- 価格表示はレンジ表示とする。
- 現時点の表示方針は「70,000円〜150,000円」。
- 提案内容は相手に応じて調整するが、サイト上の見せ方は一貫させる。
### 今後調整すること
- レンジの根拠や補足説明の有無。

### 未確定事項
- 価格に含まれる範囲（ページ数、納期、修正回数など）の明文化。

## 6. 役割分担
### 決定事項
- 私: 要件定義、設計、顧客対応、打ち合わせ、方向性整理。
- 彼女: デザイン。

### 今後調整すること
- 実装やコーディングの担当範囲。

### 未確定事項

## 7. 今回作る範囲（初期版）
### 決定事項
- 初期版として大枠だけを作る。
- ローカルで確認できる状態を最優先にする。
- 複数ページ構成の営業用サイトを作る。
- 必要なセクションを並べ、全体の導線が見える状態にする。
- 実績は最初はダミーでもよい。
- 静的サイトとして実装する。
- 使用技術は HTML / CSS / JavaScript のみとする。
- フレームワークは使わない。
- 作成対象ファイルは `index.html` / `style.css` / `script.js` とする。
- 複数ページ構成に必要な追加HTMLファイルを作成してよい。
- 必要に応じて `assets/` フォルダを用意してよい。
- 全ページに共通のヘッダー（ナビゲーション）を設置する。
- お問い合わせ導線は「フォーム＋LINE/SNS」とする（デモ表示であることを明記する）。
- 実績は「匿名/一部伏せ」で高品質ダミーとして掲載する（サンプル表記を明記する）。
- 文章は本番品質で整える。

### 今後調整すること
- 各セクションの情報量と優先順位。

### 未確定事項
- ダミー実績の具体的な内容や体裁。

## 10. デザイン方針
### 決定事項
- 信頼感。
- 洗練。
- ミニマル。
- 余白を活かす。
- スマホファースト。
- デザイン重視。
- 営業用として「頼みたくなる」ことを重視。
- 白ベースを基本にしたシンプルで見やすい構成。
- 白ベースを基本としたシンプルなデザインとする。
- 文字色は濃いグレーを使用し、可読性と落ち着きを重視する。
- アクセントカラーは深いネイビーを使用する。
- 全体のトーンは「洗練・上品・信頼感・ミニマル」とする。
- 余白を広く取り、情報を詰め込みすぎない設計とする。
- スマホファーストで設計し、スマホ表示でも美しく見えることを重視する。
- 文字を主役としたレイアウトにする。
- 実績（ポートフォリオ）は視覚的に分かりやすく、大きめに見せる。
- CTAボタンは視認性を高め、明確に目立たせる。
- 過剰な装飾、過度なアニメーション、派手なグラデーションは避ける。

### 今後調整すること
- タイポグラフィやトーンの具体化。

### 未確定事項
- カラーパレットや写真トーンの詳細。
